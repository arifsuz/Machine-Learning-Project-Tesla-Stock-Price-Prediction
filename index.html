Tesla Stock Price Prediction: Model Comparison

Perbandingan performa 3 model machine learning untuk prediksi naik/turunnya harga saham Tesla.

Decision Tree

  • Accuracy: 61%
  • Precision: Lebih tinggi untuk "Naik"
  • AUC: 0.6451

Paling baik dalam memprediksi kenaikan harga, namun kurang pada penurunan.

Naive Bayes

  • Accuracy: 52.8%
  • AUC: 0.6049

Performa sedang, kurang baik pada data tidak seimbang.

K-Nearest Neighbors

  • Accuracy: 58.3%
  • AUC: 0.6217

Lebih baik dari Naive Bayes, sensitif terhadap pemilihan k dan normalisasi fitur.

Model Comparison Chart

Kesimpulan

Decision Tree memberikan hasil terbaik untuk prediksi naik/turun harga saham Tesla pada data ini. KNN juga cukup baik setelah normalisasi. Untuk hasil lebih baik, bisa dicoba fitur tambahan atau model ensemble.
